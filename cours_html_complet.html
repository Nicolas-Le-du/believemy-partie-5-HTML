1. 📖 Introduction au HTML

HTML (HyperText Markup Language) est le langage de balisage standard pour créer des pages web. Il décrit la structure et le contenu d'une page web à l'aide de balises.

🎯 Objectif : Maîtriser HTML pour créer des pages web structurées et accessibles.

2. 🚀 Comment commencer un projet HTML ?

Étapes pour créer votre premier projet :

  1. Créer un dossier pour votre projet (ex: "mon-premier-site")
  2. Ouvrir Visual Studio Code et ouvrir ce dossier
  3. Créer un fichier index.html
  4. Installer des extensions utiles comme "Material Icon Theme" pour de jolies icônes
💡 Conseil pro : Utilisez l'extension "Live Server" dans VS Code pour voir vos modifications en temps réel !

3. 🏗️ Structure de base d'une page HTML

Voici la structure minimale que doit avoir toute page HTML :

<!-- Déclaration du type de document HTML5 --> <!DOCTYPE html> <!-- Balise racine qui englobe tout le contenu, lang="fr" pour le français --> <html lang="fr"> <!-- En-tête de la page (métadonnées, non visible par l'utilisateur) --> <head> <!-- Encodage des caractères pour les accents français --> <meta charset="UTF-8"> <!-- Responsive design pour mobile --> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- Titre affiché dans l'onglet du navigateur --> <title>Ma première page HTML</title> </head> <!-- Corps de la page (contenu visible par l'utilisateur) --> <body> <h1>Bonjour le monde !</h1> <p>Ceci est ma première page HTML.</p> </body> </html>
Résultat : Cette structure crée une page web basique avec un titre et un paragraphe.

4. 🏷️ Qu'est-ce qu'une balise ?

Une balise HTML est un élément de code entouré par des chevrons < >. La plupart des balises fonctionnent par paires :

  • Balise ouvrante : <nom-balise>
  • Contenu : Le texte ou autres balises
  • Balise fermante : </nom-balise>
<!-- Exemple de balise paragraphe --> <p>Ceci est un paragraphe</p> <!-- Exemple de balise titre --> <h1>Ceci est un titre principal</h1> <!-- Balises auto-fermantes (pas de contenu) --> <br> <!-- Retour à la ligne --> <hr> <!-- Ligne horizontale --> <img src="image.jpg" alt="Description"> <!-- Image -->

5. 🎨 Les nouvelles balises de structure HTML5

HTML5 a introduit des balises sémantiques qui donnent du sens au contenu :

<!-- Structure sémantique HTML5 --> <body> <!-- En-tête de la page/section --> <header> <h1>Mon Site Web</h1> <nav> <!-- Navigation principale --> <ul> <li><a href="#accueil">Accueil</a></li> <li><a href="#about">À propos</a></li> <li><a href="#contact">Contact</a></li> </ul> </nav> </header> <!-- Contenu principal --> <main> <!-- Article ou contenu principal --> <article> <h2>Mon premier article</h2> <p>Contenu de l'article...</p> </article> <!-- Contenu secondaire --> <aside> <h3>Sidebar</h3> <p>Informations complémentaires...</p> </aside> </main> <!-- Pied de page --> <footer> <p>© 2024 Mon Site Web. Tous droits réservés.</p> </footer> </body>
🎯 Avantages : Meilleur référencement SEO, accessibilité améliorée, code plus lisible !

6. ⚙️ Qu'est-ce qu'un attribut ?

Les attributs fournissent des informations supplémentaires sur les balises. Ils s'écrivent dans la balise ouvrante :

<!-- Syntaxe: <balise attribut="valeur">contenu</balise> --> <!-- Attribut id (identifiant unique) --> <p id="mon-paragraphe">Paragraphe avec ID</p> <!-- Attribut class (pour le CSS) --> <p class="texte-important">Paragraphe avec classe</p> <!-- Attributs d'image --> <img src="photo.jpg" alt="Description de l'image" width="300" height="200"> <!-- Attributs de lien --> <a href="https://example.com" target="_blank" title="Aller vers example.com">Lien externe</a> <!-- Attributs multiples --> <div class="container" id="main-content" style="color: blue;"> Contenu avec plusieurs attributs </div>

7. 💬 Ajouter des commentaires

Les commentaires permettent d'expliquer le code. Ils ne sont pas affichés sur la page :

<!-- Ceci est un commentaire sur une ligne --> <!-- Ceci est un commentaire sur plusieurs lignes très utile pour expliquer du code complexe --> <div class="header"> <!-- Section d'en-tête de la page --> <h1>Mon titre</h1> </div> <!-- TODO: Ajouter le footer plus tard -->
🔥 Raccourci VS Code : Ctrl + / (Windows/Linux) ou Cmd + / (Mac) pour commenter/décommenter rapidement !

9. ✨ La mise en forme du texte

HTML offre plusieurs balises pour mettre en forme le texte :

<!-- Texte en gras (importance forte pour les moteurs de recherche) --> <p><strong>Texte très important</strong></p> <p><b>Texte en gras (visuel uniquement)</b></p> <!-- Texte en italique (emphase pour les moteurs de recherche) --> <p><em>Texte avec emphase</em></p> <p><i>Texte en italique (visuel uniquement)</i></p> <!-- Autres mises en forme --> <p><u>Texte souligné</u></p> <p><mark>Texte surligné</mark></p> <p><del>Texte supprimé</del></p> <p><ins>Texte inséré</ins></p> <!-- Exposants et indices --> <p>H<sub>2</sub>O (indice)</p> <p>E=mc<sup>2</sup> (exposant)</p> <!-- Texte préformatté (conserve espaces et retours à la ligne) --> <pre> Ce texte conserve ses espaces et ses retours à la ligne </pre> <!-- Ligne horizontale --> <hr>

10. 📋 Faire des listes

HTML propose trois types de listes principales :

Liste non ordonnée (à puces)

<!-- ul = unordered list (liste non ordonnée) --> <ul> <!-- li = list item (élément de liste) --> <li>Premier élément</li> <li>Deuxième élément</li> <li>Troisième élément</li> </ul>

Liste ordonnée (numérotée)

<!-- ol = ordered list (liste ordonnée) --> <ol> <li>Première étape</li> <li>Deuxième étape</li> <li>Troisième étape</li> </ol>

Liste avec des lettres (attribut type)

<!-- type="a" pour lettres minuscules --> <ol type="a"> <li>Premier point</li> <li>Deuxième point</li> <li>Troisième point</li> </ol> <!-- type="A" pour lettres majuscules --> <ol type="A"> <li>Premier point</li> <li>Deuxième point</li> </ol> <!-- type="i" pour chiffres romains minuscules --> <ol type="i"> <li>Premier point</li> <li>Deuxième point</li> </ol>

Liste de définitions

<!-- dl = definition list (liste de définitions) --> <dl> <!-- dt = definition term (terme à définir) --> <dt>HTML</dt> <!-- dd = definition description (définition) --> <dd>HyperText Markup Language</dd> <dt>CSS</dt> <dd>Cascading Style Sheets</dd> </dl>

11. 🔗 Tout sur les liens

Les liens sont l'essence du web. La balise <a> (anchor) permet de créer des liens :

Lien externe (vers un autre site)

<!-- href = hypertext reference (référence du lien) --> <!-- target="_blank" = ouvrir dans un nouvel onglet --> <a href="https://believemy.com" target="_blank">Aller sur BelieveMyco</a> <!-- rel="noopener noreferrer" pour la sécurité avec target="_blank" --> <a href="https://example.com" target="_blank" rel="noopener noreferrer"> Lien sécurisé vers l'extérieur </a>

Lien interne (vers une section de la page)

<!-- Lien vers une ancre dans la même page --> <a href="#introduction">Aller à l'introduction</a> <!-- L'élément cible avec l'id correspondant --> <h2 id="introduction">Introduction</h2>

Lien vers une autre page du site

<!-- Lien relatif (même dossier) --> <a href="contact.html">Page de contact</a> <!-- Lien vers un sous-dossier --> <a href="pages/about.html">À propos</a> <!-- Lien vers le dossier parent --> <a href="../index.html">Retour à l'accueil</a>

Lien pour télécharger un fichier

<!-- L'attribut download force le téléchargement --> <a href="documents/cv.pdf" download="Mon_CV.pdf"> Télécharger mon CV </a> <!-- Télécharger une image --> <a href="images/logo.png" download="logo_believemy.png"> Télécharger le logo </a>

Lien pour envoyer un email

<!-- Lien email simple --> <a href="mailto:contact@example.com">Envoyer un email</a> <!-- Email avec sujet prédéfini --> <a href="mailto:louisnicolas@believemy.com?subject=Premier contact"> Contacter Louis Nicolas </a> <!-- Email complet avec destinataires multiples --> <a href="mailto:contact@example.com?subject=Demande d'information&cc=info@example.com&bcc=admin@example.com&body=Bonjour,%0D%0A%0D%0AJe souhaiterais...%0D%0A%0D%0ACordialement"> Email complet </a>

Lien avec infobulle (tooltip)

<!-- L'attribut title affiche une infobulle au survol --> <a href="https://believemy.com" title="BelieveMyco - Plateforme de formation en ligne" target="_blank"> BelieveMyco </a>

Exemples de liens :

  • Lien interne avec tooltip
  • Lien email
  • Lien externe
🎯 Paramètres email :
• subject = Sujet de l'email
• cc = Copie carbone (visible)
• bcc = Copie carbone invisible
• body = Corps du message
• %0D%0A = Retour à la ligne dans l'URL

🎯 Bonnes pratiques pour les emails HTML :

  • Utilisez des tables pour la mise en page (les divs ne fonctionnent pas partout)
  • CSS inline uniquement - pas de feuilles de style externes
  • Largeur maximum 600px pour une bonne lisibilité
  • Images hébergées en ligne avec attribut alt obligatoire
  • Testez sur plusieurs clients (Gmail, Outlook, Apple Mail, etc.)
  • Version mobile-friendly avec media queries si possible
  • Lien de désinscription obligatoire (RGPD)
⚠️ Limitations des emails :
• Pas de JavaScript
• CSS limité (pas de flexbox, grid, etc.)
• Support inconsistant selon les clients
• Images souvent bloquées par défaut

🔧 Outils recommandés pour les emails :

  • MJML - Framework pour emails responsive
  • Litmus - Test sur différents clients email
  • Email on Acid - Tests et prévisualisation
  • Can I Email - Compatibilité des propriétés CSS
